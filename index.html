CMD Homebase is de plek op Github waar educatief materiaal en code voorbeelden beheerd en gearchiveerd wordt voor de technieklessen van CMD. De tutorials en voorbeelden zijn praktisch en toegespitst op datgene wat aangeleerd wordt tijdens de lessen, en kunnen als referentie of uitgangspunt gebruikt worden door studenten die verder op de technieken willen bouwen. Homebase is een initiatief van CMD op Github en ontwikkeld door David van den Bor.
